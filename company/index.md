# 식자재쿡 · 브랜드쿡
{: .no_toc }

**AI 기반 파트너십 유저체인을 활용한 프랜차이즈 및 외식업 전용 통합 오퍼레이팅 플랫폼**
{: .fs-6 .fw-300 }

㈜식자재쿡(2024년 설립, 부산)은 외식업 창업 준비부터 매장 운영 전 과정을 아우르는 End-to-End 플랫폼을 운영합니다. 유통 법인(식자재 공급·물류)과 IT 법인(플랫폼·AI 개발)의 양대 축으로, 모바일 앱 2종(식자재쿡·브랜드쿡)과 프랜차이즈 가맹본부 전용 웹 대시보드를 제공합니다. 홈페이지: [www.식자재쿡.com](https://www.식자재쿡.com)

---

## 1. 우리가 보는 문제

- **자영업 시장의 불안정성** — 외식업 폐점률이 신생률을 상회하고, 음식점업 1년 생존율은 65.9% 수준. 체계적 관리가 부족한 소규모 가맹본부와 초보 창업자의 경영난이 심화되고 있습니다.
- **창업 준비의 파편화** — 설비·인테리어·POS·마케팅 등 준비 과정이 쪼개져 있어 예비 창업자는 비용 절감에 한계를 느낍니다.
- **데이터 기반 관리 부재** — 신생 프랜차이즈는 식자재 구매력(Buying Power)이 약하고, 전용상품 재고·QSC 관리에 취약합니다. 높은 3PL 물류수수료 부담은 결국 가맹점주에게 전가됩니다.

## 2. 솔루션

### 식자재쿡 — 외식업 통합 식자재 플랫폼
"식자재 유목민들을 위한 외식업 통합솔루션." 부산·경남·경북 **최소주문금액 0원, 배송비 0원**, 서울·수도권 최소주문 3만 원. 야채 경매권 보유로 **당일 경매·당일 수매** 직소싱, 콜드체인 광역 물류, 4만 가지 이상 상품 구성, 100% 상품 교환 A/S를 제공합니다.

### 브랜드쿡 — 프랜차이즈 전용 폐쇄몰
가맹본부·점주 상생을 목표로 하는 프랜차이즈 전용 플랫폼. 전용(PB·MPB)/범용(NB) 상품 발주현황 대시보드, 매장 POS 매출연동, 레시피 기반 자동발주로 **사입 방지와 브랜드 품질 일관성**을 지원합니다. 브랜드별 UI 커스터마이징을 제공합니다.

### AI 에이전트 '쿡'
레시피 표준화 기반 최적 식자재·적정 발주량 제안, 구매 패턴 분석 스마트 발주, 공공 API 시세 데이터 기반 신선상품 매입 타이밍 제안, AI 비대면 QSC 점검까지 — 데이터 기반 운영 최적화를 담당합니다.

### 유저체인 (전략적 비즈니스 얼라이언스)
주방집기·인테리어·공조·기기렌탈·마케팅 등 검증된 협력사 풀을 플랫폼에 내재화해 창업 준비 시간을 단축하고, 협력사 회원 대상 교차 마케팅으로 사용자 유입 시너지를 만듭니다. 신규 창업자에게는 '창업 AtoZ 패키지'로 초기 Lock-in을 강화합니다.

## 3. 경쟁 포지션

식자재를 **직접 구매·유통**하며 가맹사 전용상품을 직접 취급하고, 동시에 가맹본부·점주용 데이터를 직접 수집·분석·제공하며, 협력사 유저체인으로 네트워킹을 활성화하는 플랫폼은 경쟁 시장 내에서 식자재쿡이 유일합니다. (자산 보유형 유통 + 데이터 SaaS의 결합 — 자산 경량 SaaS 경쟁사와의 근본적 차별점)

## 4. 수익 모델

프랜차이즈 PB·NB 상품 판매(식자재 공급 수익), 브랜드쿡 앱 이용(판매) 수수료, 가맹점 수 기반 SaaS 솔루션 이용료(기능별 플러그인 차등 과금)로 수익 구조를 다변화하고 있습니다.

## 5. 성장 전략

- **파트너 생태계 확장** — 외식업계 선두 인프라 파트너(POS·테이블오더, 자영업자 커뮤니티 등)와의 연계로 매장 운영 필수 시스템을 플랫폼 안에서 즉시 구축.
- **지역 물류 인프라** — 로컬 파트너사 ERP 연동 기반 지역 밀착형 물류망, 배송기사 전용 앱으로 서비스 품질 고도화.
- **AI 고도화** — RAG 기반 가맹점별 특화 답변 생성, 수요예측·재고보충 알고리즘(LSTM 등) 구현과 특허 출원 지속.
- **가맹사업 인큐베이팅** — 영남권 유망 중소 외식 브랜드의 프랜차이즈 가맹사업화 지원(직영 인큐베이팅 브랜드 '수국칼' 운영).
